RAG Document QA — browser demo against the live /ask endpoint.

Pick an example or type a question against the eight-document policy corpus. Answers and cited sources come from the live API — useful when reviewers want evidence, not slides.

  • FAISSvector retrieval

Ask the demo corpus

Fixed index: 8 synthetic policy documents (SLA, billing, refunds, incident response, support workflow, onboarding, pricing, internal FAQ) from data/raw_docs/. Example chips above map to those themes.

Calls POST /ask on the deployed API from your browser. Responses include answer and sources (document, page, chunk_id). If the request is blocked (CORS or API key on the server), use API docs or curl instead.
